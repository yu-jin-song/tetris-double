# Tetris
- 고전 게임 `테트리스`를 구현
<br>

# How To
### ✅ 조작법
<img src="https://user-images.githubusercontent.com/74666378/113044241-63e75300-91d8-11eb-9dd4-b82b7fe28cab.jpeg"><br>
- 왼쪽으로 이동 : `A` `←`
- 오른쪽으로 이동 : `D` `→`
- 블록 변경 : `W` `↑`
- 속도 증가 : `S` `↓`
<br>
